Agora Grid

Write less to support all screens

Proportional

Indicates which percentage of the available screen should occupy each element using classes such as .mw-ui-one-half or mw-ui-two-thirds.

Responsive

For each screen you want to support, you can indicate how elements will adapt. An initial set of breakpoints are defined (@palm, @lap,@portable, and @desk) but you can define as many screen sizes as you need.

Semantic

It's based on LESS to keep the HTML free from grid classes, and allowing you to adjust any property beyond layout when being responsive.

For example, if you want some sections to occupy ⅓ by default, but to expand to the whole width on a mobile device, you need the following LESS code:

section { .mw-ui-item; .mw-ui-one-third; @media @palm { .mw-ui-one-whole; font-size: 2.2em; } }
Extensions of the grid system allow it to add explicit media proportions or use the flexbox model.
